🔑 批量导入关键词

📖 字段说明（点击折叠）
字段必填说明
基础字段
pattern ✓ 必填 关键词文本 / 正则表达式
match 可选 匹配方式：contains / exact / regex，默认 contains
mode 可选 触发模式：random=随机一条 / all=全部发送，默认 random
回复内容（同一 pattern 可用多行，合并为多条回复）
reply_type 可选 回复类型：text / photo / video / audio / document / animation / voice / sticker，默认 text
reply_text 条件必填 文本内容（text 类型必填；支持 HTML：<b> <i> <a> 等）
reply_file_id 条件必填 Telegram file_id（非 text 类型必填，可从 Bot 发送媒体获取）
reply_caption 可选 媒体说明文字（photo/video/document 等可用，支持 HTML）
自动删除回复（两种写法二选一）
delete_after_seconds 可选 发送后 N 秒自动删除；0 或空=不自动删除
delete_after_value
delete_after_unit		 可选 便捷写法：数值 × 单位秒数。如 value=10, unit=60 = 10 分钟
单位参考：1=秒 60=分 3600=时 86400=天
（若同时填了 _seconds，则以 _seconds 为准）
自动到期（关键词本身）
expire_after_seconds 可选 N 秒后自动停用该关键词；空=永不过期
expire_after_value
expire_after_unit		 可选 便捷写法，同上。示例：value=7, unit=86400 = 7 天后到期
生效范围和时间
start_at 可选 开始生效时间："YYYY-MM-DD HH:MM:SS" 或 "YYYY-MM-DD HH:MM"；空=立即生效
chat_ids 可选 监控哪些群组，留空 = 所有群组
CSV/Excel：逗号分隔，如 -1001,-1002
JSON：可用数组，如 [-1001, -1002]
active 可选 1=启用（默认） / 0=停用

⏰ 批量导入定时任务

📖 字段说明（点击折叠）
字段必填说明
基础字段
name ✓ 必填 任务名称（用于显示和日志）
chat_id ✓ 必填 目标 Chat ID，如 -1001234567890
触发时机
once 可选 0=周期执行（默认） / 1=一次性任务
cron ✓ 必填 周期任务（once=0）：5 字段 cron 表达式
  0 9 * * * = 每天 9 点
  */30 * * * * = 每 30 分钟
  0 9 * * 1-5 = 工作日 9 点
  0 0 1 * * = 每月 1 日 0 点
一次性任务（once=1）：具体时间 "YYYY-MM-DD HH:MM:SS"
消息内容
msg_type 可选 消息类型：text / photo / video / audio / document / animation / voice / sticker，默认 text
msg_text 条件必填 text 类型必填；支持 HTML 富文本
msg_file_id 条件必填 非 text 类型必填
msg_caption 可选 媒体说明（photo/video/document 等可用）
自动删除消息
delete_after_seconds 可选 发送后 N 秒自动删除；0 或空=不自动删除
delete_after_value
delete_after_unit		 可选 便捷写法：数值 × 单位秒数。示例：value=30, unit=60 = 30 分钟
单位参考：1=秒 60=分 3600=时 86400=天
状态
active 可选 1=启用（默认） / 0=停用
